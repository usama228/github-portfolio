AMS

Attendance Management System

The MERN Stack Attendance Management System is a web-based application designed to efficiently track employee attendance with real-time check-in/check-out functionality. Built using MongoDB, Express.js, React.js, and Node.js, with PostgreSQL as the database, this system provides an intuitive dashboard for managing attendance records, leave requests, and work-hour analytics. The system includes secure authentication with Auth0, role-based access control for Admins, Employees, and Team Leads, and a fully responsive UI built with Bootstrap. Employees can check in/out, track break durations, and submit leave requests (Sick Leave, Casual Leave, Urgent Leave). For backend efficiency, the system uses Node.js and Express.js, with PostgreSQL ensuring structured and scalable data management.
